General overview - top

This is an interactive template of the face anatomy that consists of HTML, CSS, JS files (Also include installable WordPress plugin). You can customize the hover information, links, etc through simple JavaScript files (JavaScript knowledge is not required). Also you can add unlimited number of clickable spots anywhere on the figure using X, Y coordinates, then embed the customized figure in your website pages.

Installation in a Standard website - top

  1. Upload the folder 'anatomy-face' to any directory in your website.
    You can use FileZilla as FTP client to upload files to your website, and here is a video tutorial about that.
  2. Test the template by opening the full path of the 'index.html' file in a web browser (such as Google Chrome), the full path should be something like this:
    http://www.yoursite.com/anatomy-face/index.html
    If you don't see the interactive figure or redirected to '404 Page not found', make sure you are using the correct path of the uploaded folder.
  3. Copy the following code and paste it before </head> tag in your website page, and make sure to change 'yoursite.com' to the correct full path of the files. 
    <link href="http://www.yoursite.com/anatomy-face/illustration-style.css"  rel="stylesheet" type="text/css"/>
<script src="http://www.yoursite.com/anatomy-face/jquery.min.js"></script>
<script src="http://www.yoursite.com/anatomy-face/anatomy-settings.js"></script>
<script src="http://www.yoursite.com/anatomy-face/spots-settings.js"></script>
<script src="http://www.yoursite.com/anatomy-face/interact-script.js"></script>
  4. On your computer, open the file 'anatomy-face/index.html' using any code editor (such as Sublime or Notepad++).
  5. Copy the code in between <body> and </body>
    from 
    <span id="anatip"></span>
<div id="anawrapper">
    down to... 
    <div class="clear"></div>
  6. Paste the code in your website page.
Important note: With some website templates and due to divs arrangement, the floating hover popup may be pushed away from where it should be, to avoid that we recommend that you cut the span "anatip" line of code and paste it just after the opening body tag of the page like this: 
<body>
<span id="anatip"></span>

Installation in a WordPress website - top

Inside the folder 'WordPress Plugin' you will find the installable WordPress plugin zip file 'anatomy-face.zip', you need to login to your WordPress website admin panel and install the plugin as follow:

  1. Go to 'Plugins' > 'Add New'
  2. Click 'Upload Plugin' then 'Choose File', on your computer select the file 'anatomy-face.zip' ( included ).
  3. Click 'Install Now', then 'Activate Plugin'.
  4. In the left side bar you will see a new tab for the Anatomy figure, click it to start the customization.

You will be able to customize the hover information and the link for each part through the visual editor. However adding clickable spots needs to be done by manual editing of the plugin file "spots-settings.js" and you can access this file through your WordPress admin panel under "Plugins" > "Editor".

How to open a modal window on part click?
  1. Install Popup Maker plugin.
  2. Click "Add New Popup" and write the wanted information.
  3. Then click the button "Add New Trigger" > "Click Open" > "Extra CSS Selector".
  4. Write dot + the default part name as seen in the dashboard (but use lower case and replace spaces with dash), examples:
    FULL FACE > .full-face
    FOREHEAD > .forehead
    EYE [RT] > .eye-rt
    CHEEK [LT] > .cheek-lt
  5. Click "Add" then "Publish".
  6. In the anatomy dashboard, for that part choose "Target" > "None / Modal" instead of the default "Same Page".
    7. Note: The modal window will work in the frontend pages not in the dashboard preview.

Customization? - top

For the Standard template, you can customize the hover information, link, etc for each part through the included 'anatomy-settings.js' file (JavaScript knowledge is not required). You can open that file using any code editor (such as Sublime or Notepad++).

  • "hover": "write some basic information for the popup",
  • "url": "http://www.thelink.com/",//link to any webpage
  • "target": "same_window",
    1. "same_window" to open the link in the same window.
    2. "new_window" to open the link in a new window.
    3. "modal" to open a modal window, in this case you need to change the above "url" to the required modal window name such as
      "url": "#mymodal"
      Note: This option works only with Bootstrap website templates and Bootstrap based WordPress themes, if you are not sure or if you don't know how to use it, you need to ask your website template/theme provider.
    4. "none" use it if you have enough information in the popup and you don't want to make the area clickable but the popup still working.
  • "enabled": true
    You can set it to false to make this area inactive (not clickable, no highlight, no popup).

How to add/remove and modify spots? - top

You can customize the existing pins, add or remove pins through the file 'spots-settings.js'.

  • "hover": "write some basic information for the popup",
  • "pos_X": 20, "pos_Y": 50, use this guide to obtain the wanted values.
  • "size": 18, size of the spot.
  • "outline": "#660000".
  • "upColor": "#e60000", "overColor": "#ffd480".
  • "url": "http://www.thelink.com/",//link to any webpage
  • "target": "same_window",
    1. "same_window" to open the link in the same window.
    2. "new_window" to open the link in a new window.
    3. "modal" to open a modal window, in this case you need to change the above "url" to the required modal window name such as
      "url": "#mymodal"
      Note: This option works only with Bootstrap website templates and Bootstrap based WordPress themes, if you are not sure or if you don't know how to use it, you need to ask your website template/theme provider.
    4. "none" use it if you have enough information in the popup and you don't want to make the area clickable but the popup still working.
  • "enabled": true
    You can set it to false to make this area inactive (not clickable, no highlight, no popup).
